projects

WHITEBOARD

For my second project at the DALI Lab (Spring & Summer 2020), my team designed and developed a calendar platform called Whiteboard that helps individuals plan their lives. The web app is built using React and Cloud Firestore and integrates a variety of personal organization features including project and task list creation, habit tracking, and schedule planning. The project is currently in beta testing.

TINY SEARCH ENGINE

I used C to develop a tiny search engine for one of my computer science classes at Dartmouth. The search engine consists of a crawler, indexer, and querier. The crawler crawls the web from a given seed webpage up to a given max depth and caches the content of the pages it finds. The indexer reads cached files created by the crawler and builds an index that maps from words to pages. The querier performs search queries consisting of words combined by 'and' & 'or' and uses the crawler/indexer-produced files to output a ranked list of webpages that match each search query.

DARTMOUTH D-PLAN

For my first project at the DALI Lab (Winter 2020), my team designed and developed a new online portal for Dartmouth students to change their enrollment patterns (their D-Plans). We worked with the Dartmouth Registrar and Dartmouth Information, Technology, & Consulting (ITC) to make the new system more inclusive of different student groups and provide more guidance to students when making important academic decisions.

OPEN ENERGY HUB BACKEND

As part of my study abroad program in summer 2019, I did a short, unpaid internship at BlockLab NL. My internship group developed a PoC blockchain backend system for BlockLab's Open Energy Hub project using JavaScript, Node.js, 3Box, IPFS, and OrbitDB. For this project, I did a lot of reading to learn about blockchain technology, distributed databases, and the technologies that we were using to build the backend.

MOCK AIRBNB MOBILE APP

For the final project of my Full Stack class (Summer 2019), I built a mobile app inspired by Airbnb. The app allows guests to register an account, login/logout, view listings, and create bookings; all of this information is stored in a MySQL database. For this project, I worked with HTML, CSS, JavaScript, TypeScript, Ionic, Angular, Node.js, Express, Postman, and MySQL.

In my first year at Dartmouth, I did research as a Women In Science Project intern with Professor Chris Bailey-Kellogg. For my project, I used computational methods (AutoDock Vina, PyMol, Python) to analyze potential antibiotic enzymes against drug-resistant Staph.
